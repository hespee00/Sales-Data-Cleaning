# Sales Data Cleaning with Power Query

## Project Overview
This project focuses on cleaning and transforming a messy sales dataset into a structured, analysis-ready format using Power Query in Microsoft Excel.

The dataset was originally in a wide format, making it difficult to analyze efficiently. This project demonstrates how to restructure and improve data quality for better analytical outcomes.

## Objective
To clean and transform the sales dataset to ensure it is reliable, consistent, and suitable for analysis.

## Dataset
- Source: [Foresightbi](https://foresightbi.com.ng/microsoft-power-bi/dirty-data-samples-to-practice-on)
- Type: Sales Data  
- Size: 835 rows  

## Tools Used
- Microsoft Excel  
- Power Query  

## Data Cleaning Process

The following steps were performed using Power Query:

- Extracted the data using Power Query Editor   
- Transformed data from wide format to a normalized format (unpivoting)  
- Removed 3 unnecessary columns to improve clarity
- Replaced null cells with appropriate values
- Standardized column names for consistency    
- Validated dataset for accuracy and completeness  

## Key Outcomes

- Transformed unstructured data into an analysis-ready format    
- Established a repeatable data cleaning workflow
